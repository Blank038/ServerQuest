# ServerQuest
## 介绍
基于 Bukkit 开发的全服任务系统, 额外扩展 PixelmonReforged 模组。
![](./files/example.png)

## 功能
* 界面的高度自定义性
* 高定制性的文本词条
* 全服进度要求/个人进度要求
* JavaScript 语法条件(支持 PlaceholderAPI)

## 更新计划
- [ ] 支持 MySQL/SQLite 存储数据
